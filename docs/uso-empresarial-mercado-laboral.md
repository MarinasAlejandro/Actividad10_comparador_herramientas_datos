|---------|----------|---------|

### Salarios Medios (USA, referencia 2024-2025)

| Nivel | Power BI | Tableau |
|-------|----------|---------|
| **Junior / Analyst** | $55,000 - $75,000 | $60,000 - $80,000 |

**Insight:** Tableau tiende a pagar ligeramente más porque se asocia a roles más especializados en analytics y data science. Power BI se pide más frecuentemente pero como skill complementaria (no como rol principal).

### Tendencias de Crecimiento

- **Power BI:** Crecimiento acelerado impulsado por la omnipresencia de Microsoft 365. La integración con Copilot (IA generativa) impulsa nueva ola de adopción en 2025-2026.
- **Tableau:** Crecimiento más moderado pero sostenido. La adquisición por Salesforce abrió el mercado CRM-analytics. Tableau Pulse (IA) y la integración con Slack redefinen su propuesta de valor.
- **Tendencia convergente:** Ambas herramientas integran cada vez más IA generativa (NLP para consultas, insights automáticos, narrativas generadas).

## 3. Ecosistema y Efecto de la Matriz Corporativa

### Power BI — Ecosistema Microsoft

```
Microsoft 365 ←→ Power BI ←→ Azure Synapse / Fabric
     ↕                              ↕
Excel, Teams,                 Data Lake, ML,
SharePoint, OneDrive          Databricks integration
     ↕
  Copilot (IA generativa)
```

**Ventajas del ecosistema:**
- Incluido parcialmente en licencias Microsoft 365 E5
- Integración nativa con Excel (el 80% de analistas ya usa Excel)
- Publicación directa en Teams y SharePoint
- Azure como backend de datos escalable
- **Microsoft Fabric** (2023-2024): plataforma unificada de datos
- **Copilot en Power BI**: generación de DAX, narrativas automáticas y Q&A en lenguaje natural

**Impacto en adopción:** Muchas empresas adoptan Power BI "por defecto" porque ya pagan Microsoft 365.

### Tableau — Ecosistema Salesforce

```
Salesforce CRM ←→ Tableau ←→ Tableau Cloud / Server
      ↕                            ↕
Sales Cloud,                  Prep Builder,
Service Cloud,                Tableau Pulse (IA),
MuleSoft                      Slack integration
      ↕
  Einstein AI / Data Cloud
```

**Ventajas del ecosistema:**
- Integración directa con Salesforce CRM (datos de ventas, clientes, pipelines)
- **Tableau Pulse** (2024): métricas personalizadas con IA que detectan anomalías
- Integración con Slack para compartir insights
- MuleSoft para conectar fuentes de datos heterogéneas
- Comunidad técnica muy activa (Tableau Public, Tableau Community Forums)
- Superior en capacidades de visualización y exploración ad-hoc

**Impacto en adopción:** Empresas con Salesforce CRM tienen un path natural hacia Tableau. Fuera de ese ecosistema, compite por méritos técnicos pero con precio más alto.

## Conclusiones clave para slides

1. **Power BI gana en volumen y accesibilidad.** Es la herramienta BI más adoptada del mundo, impulsada por el ecosistema Microsoft y un precio agresivo.

2. **Tableau gana en profundidad analítica y visualización.** Sigue siendo el estándar de referencia en capacidades de exploración visual de datos. Roles más especializados y salarios ligeramente superiores.

3. **No es "una u otra" — es contexto.** La elección depende del ecosistema existente (Microsoft vs Salesforce), presupuesto, nivel de sofisticación analítica y tamaño de empresa.

### Para el mercado laboral español:
- Power BI domina en ofertas de trabajo en España (ratio ~3:1 vs Tableau)
- La certificación PL-300 de Microsoft es probablemente la certificación BI con mejor ROI en España
- Saber ambas es el perfil ideal, pero si hay que elegir una: **Power BI** por demanda, **Tableau** para diferenciarse en roles analíticos avanzados
